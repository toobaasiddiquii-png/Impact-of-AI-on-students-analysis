#  AI Impact on Students Analysis

##  Project Overview

This project analyzes survey data on the impact of Artificial Intelligence (AI) on students using Microsoft Excel. The analysis focuses on identifying trends in AI usage, study habits, academic performance, and student perceptions through data cleaning, Pivot Tables, and data visualization.

The project demonstrates how Excel can be used to transform raw survey data into meaningful insights and present them through organized charts and summary tables.



##  Objectives

- Clean and prepare the dataset for analysis.
- Explore AI usage patterns among students.
- Analyze relationships between AI usage and academic performance.
- Summarize findings using Pivot Tables.
- Visualize insights using Excel charts.



##  Tools & Technologies

- **Microsoft Excel**
  - Data Cleaning
  - Excel Functions
  - Pivot Tables
  - Pivot Charts
  - Data Visualization

##  Dataset

**Source:** Kaggle – AI Impact on Students Dataset

The dataset contains information related to:

- Student demographics
- AI usage
- Study habits
- Academic performance
- Student opinions regarding AI



##  Data Preparation

The dataset was cleaned in Microsoft Excel before analysis.

The cleaning process included:

- Removing duplicate records
- Handling missing values
- Standardizing data formatting
- Preparing the dataset for Pivot Table analysis



##  Data Analysis

The analysis was performed using Pivot Tables and Pivot Charts to summarize key trends in the dataset.

The project explores:

- AI usage among students
- Academic performance
- Study patterns
- Student perceptions of AI
- Comparisons across different groups



##  Visualizations

The project includes multiple Pivot Charts that summarize the analysis and present the findings in a clear and organized manner.
